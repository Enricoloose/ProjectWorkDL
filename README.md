# ProjectWorkDL
Project Work Deep Learning - Text classification su dataset matematico

## Obiettivo progetto
Sviluppare un modello per andare a fare classificazione su sette label di un dataset composto da esercizi matematici

## Struttura Progetto
Ogni cartella indica un modello sviluppato. project_info.md è un file che ho utilizzato come registro per memorizzare i vari cambiamenti e processi di fine-tuning
